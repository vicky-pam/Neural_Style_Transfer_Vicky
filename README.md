Project ini melakukan style transfer menggunakan model VGG19.  
Gambar content dan style digabungkan untuk menghasilkan gambar baru dengan bentuk dari content dan tekstur dari style.

## File
- `fotobangunan.jpg` — gambar content  
- `gambarabstrak.jpg` — gambar style  
- `output.jpg` — hasil style transfer  
- `tugasdll.py` — script utama
